MobileCMS - бесплатная CMS для мобильных сайтов с открытым исходным кодом.

Основные преимущества:
- Объектно ориентированная сущность
- MVC архитектура
- Модульная система
- Устойчивость к нагрузкам
- Гибкость, расширяемость
