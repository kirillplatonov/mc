Добавление виджета на главную страницу

Описание

Если вы хотете чтобы на главной сайта отображался виджет модуля (если таковой имеется), произведите следующие действия:

  • Перейдите в Панель управления -> Главная страница (или просто Главная)
  • В нужном блоке нажмите кнопку "Добавить виджет"
  • Укажите заголовок виджета, выберите модуль из списка и нажмите кнопку "Добавить"
  • Если необходимо, настройте виджет, нажав кнопку "Настроить" напротив виджета
