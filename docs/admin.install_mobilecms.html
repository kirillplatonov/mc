Установка MobileCMS

Описание

Для установки MobileCMS выполните следующие действия

  • Распакуйте архив со скриптом
  • Загрузите извлеченные файлы на сервер
  • Создайте базу MySQL для скрипта
  • Перейдите на свой сайт и действуйте согласно инструкциям инсталлятора
